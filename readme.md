# Aqui se inicia o projeto. 


O intuito é que em todos os dias úteis sejam criados projetos, pessoais, seguindo livros ou tutoriais no youtube. 
## regras:

 O nível de dificuldade aumentará gradativamente, de modo a fortalecer os conteúdos e conhecimentos adquiridos. 

 Vamos passar de HTML, CSS, até React, posteriormente chegando em backend.
O objetivo é, além de seguir com a proposta de 1 projeto por dia, é de conseguirmos criar um projeto fullstack de ponta a ponta
